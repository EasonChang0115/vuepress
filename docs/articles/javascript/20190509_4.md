## 箭頭函式

### 語法簡短
* 參數只有一個可以省略小括號
* 內容只有一行並回傳可以接為一行

```javascript=
// ES5 
var double = function(x){
    reutn x * 2;
}

// ES6 可直接在裡面做換行的動作
const double = (x) => {
   return x * 2;
}

// 若只有一個參數，0個或兩個以上就要小括號
const double = x => {
   return x * 2;
}

// 若函數本體只有一行，可以直接接在箭頭後面
const double = x => x * 2;
```
常用的array.map
```javascript=
arr.map((ele,index) => ele + index);
```
常用在監聽事件中
```javascript=
btn.addEventListener('click',()=>console.log('123'));
```

### 箭頭函式注意事項

1. 自動綁定外層的this
2. 無法用bind、apply、call來覆蓋已經被綁定的this
3. 沒有argument參數
4. 不能當作建構子使用
