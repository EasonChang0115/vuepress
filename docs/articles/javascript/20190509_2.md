## 解構賦值 Destructuring assignment

### 陣列解構
當我們想要把一個陣列裡面的值都取出來的時候，在ES5之前都必須這樣做

```javascript=
const nums = [1, 2, 3];
const first = nums[0];
const second = nums[1];
```

但在ES6後多了陣列解構的語法，我們可以用類似於鏡射的方式把值取出來。(依序取出)

```javascript=
const nums = [1, 2, 3];
// 可以改寫成
const [fitst, second, third] = nums;
```
此時first = 1、second = 2。同時也多了一些方便的功能

1. 若取出來的值不存在，可以給予預設值

```javascript=
const [fitst, second, third, four = 0] = nums;
```

2. 省略不想取出的元素，我們用逗號( , )隔開。
```javascript=
const [, second, third] = nums;
const [,, third2] = nums;
```

3. 剩餘的部分我們也可以來處理
```javascript=
const [first, ...others] = nums;
// first = 1, others = [2,3]
```
4. 變數交換
```javascript=
// 變數交換
let a = 1;
let b = 2;
[a,b] = [b,a]
```

5. 字串拆解

```javascript=
let str = '鐵人賽真的太好玩啦';
let [a, b, c, d, e, f, g, h, i] = str;
```

### 物件解構

物件解構跟陣列解構差別在於物件解構並不是依序鏡射到變數上的。物件解構是對應到key值，key值相同的才會給value，因此物件解構是沒有省略的部分，也多了一個重新命名的方式。

```javascript=
const point = {
    x: 100,
    y: 150,
    z: 200
};
const {x, y, z} = point;

// 預設值
const {x, y, z = 0} = point;

// 指派新名稱(重新命名，陣列解構是沒有的)
const {x: px, y, z} = point;

// 剩餘部分
const {x, ...other} = point;

```
#### 解構函數式參數(這很常用到)

```javascript=
function distance(point){
    return Math.sqrt(point.x*point.x + point.y*point.y);
} 

// 可以對Point解構
function distance({x, y}){
    return Math.sqrt(x*x +y*y);
}

```
