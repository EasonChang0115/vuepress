## 字串模版 String template
常常在開發時，需要組字串。但是往往太長的字串組合，到後來不好看懂也不好修改。ES6就提供了字串模板的方式，讓字串組起來更簡單，而且更好閱讀。

### 插入表達式
```javascript=
// ES5 用加號把字串連結起來
function greet(name){
    console.log("hello " + name + "!");
}
```
```javascript=
// ES6 使用模板，挖洞給變數填入，並可以使用簡單的表達式(運算)
function greet(name,days){
    console.log(`hello, ${name}! It's been ${days * 24} hours`);
}
```
用‵ ‵(backtick)包起來的字串，創造${  }模板給對應變數放入，也可以放入簡單運算。


### 多行字串
```javascript=
// ES5 多行字串
let words = "aaaaaaaaaaaaaa\n" + 
    "bbbbbbbbbbb\n" + 
    "ccccccccc";

// ES6 可直接在裡面做換行的動作
let words = `
aaaaaaaaaaa
bbbbbbbbbbbb
ccccccc
`
```
